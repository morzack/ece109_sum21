Changelog

version 1.2.5 (r436)
  • Proper clearing of the PennSim video output when the "reset" command is issued.
version 1.2.4 (r434)
  • Added "check cumulative" command to query the number of checks since the last "check reset" command was issue. The simulator resets the non-cumulative counts, but not the cumulative counts.
  • Added more useful feedback for some uncommon hardware events (memory protection fault, for example)
version 1.2.3 (r427)
  • Prevented user from trying to load object files that didn't end in .obj (a common mistake)
version 1.2.2
  • Added SUB instruction
